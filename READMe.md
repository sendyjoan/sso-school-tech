## Cara Instalasi
1. Clone repositori:
    ```bash
    git clone https://github.com/username/sso-school-tech.git
    ```
2. Masuk ke direktori proyek:
    ```bash
    cd sso-school-tech
    ```
3. Instal dependensi:
    ```bash
    npm install
    ```
4. Konfigurasi file environment `.env`.
5. Jalankan aplikasi:
    ```bash
    npm start
    ```
